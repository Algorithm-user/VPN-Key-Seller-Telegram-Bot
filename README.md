Анонимный telegram бот для автоматической выдачи ключей (Vless, Shadowsocks и т.д.) после оплаты

Пробный период - выдача одного пробного ключа по нажатию кнопки
Покупка- выдача ключа после отправки чека оплаты (ссылка формата `t.me/send?start=`)



1) Клонируйте репозиторий или скачайте файлы проекта

2) Установите зависимости:

pip install Node.js 

pip install npm

npm install

4) Создайте бота в Telegram через @BotFather и получите токен

5) Отредактируйте .env:

token=токен бота от @BotFather 

db=путь к файлу базы данных (по умолчанию ./data/keys.db)

app=название приложения для подключения (например: neko box и т.д.)

guide=ссылка на инструкцию по установке (необязательно, советуется использовать telegraph). Если указана, будет добавлена кликабельная ссылка в сообщениях с ключами

5) Добавление ключей перед запуском

   Перед первым запуском необходимо добавить ключи в базу данных.

   Создайте текстовые файлы с ключами (по одному ключу на строку):

   data/trial-keys.txt:

   vless://

   ss://

   trojan://

   data/paid-keys.txt:

   vless://
  
   ss://

   trojan://

Бот поддерживает любые типы ключей: VLESS, Shadowsocks, Trojan, VMess и другие.

6) Добавление пробных ключей:

   cmd:

   cd путь к папке проекта

   npm run add-trial data/trial-keys.txt

7) Добавление платных ключей:

   cd путь к папке проекта

   npm run add-paid data/paid-keys.txt

   просмотр статистики:

   npm run stats

Каждый ключ должен быть уникальным. Дубликаты будут игнорироваться.

8) Запуск бота:

   cmd:

   npm start


При запуске вы увидите статистику ключей и сообщение о том, что бот запущен

9) Использование бота:

   1) Найдите вашего бота в Telegram по имени

   2) Нажмите /start для начала работы

   3) Выберите действие:

   Пробный период - получить бесплатный тестовый ключ
   Купить - приобрести ключ за 5$
   
   4) При покупке отправьте чек cryptoboy(@send) на указанную сумму (ссылку формата `t.me/send?start=...`)

   5) Получите ключ

   6) Если указана ссылка guide, бот покажет инструкцию по установке

   10) Логирование

Бот автоматически записывает все действия пользователей в файл logs/bot.txt

Логи содержат информацию:

   1 Дату и время действия

   2 Тип действия (TRIAL или PAID)

   3 Username пользователя

   4 ID пользователя

   5 Имя и фамилию

   6 Ссылку на чек (для покупок)
